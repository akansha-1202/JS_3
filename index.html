QUESTION 1.  Find the number of digits

Take a number from the user and print the count of digits in that number.

SOLUTION. 
const Find_Digits = (N) => 
      {
        let count=0;
        while(N!==0){
          count++;
          N=Math.floor(N/10);
        }
        return count;
      };
Input: 1234
Output:  4


QUESTION 2.  Find the Fives.

Among all the digits from 0−9, PrepBuddy likes number 5. He has a number and wants you to find out the number of times 5 occurred in the given number.

SOLUTION. 
const Find_Five = (N) => 
      {
         let count=0;

          if(N%10==5){
          count++;
        }
        N=Math.floor(N/10);
        }
        return count;
      };

Output: 2

QUESTION 7.  Check whether a Number is a prime or not

Write a program which takes a number from user and check whether number is prime number or not a prime number.

SOLUTION. 
        const Prime_Check = (N) => 
        {
          let i; count=0;
          for(i=1;i<=N;i++){
            if(N%i===0){
              count++;
            }
          }
          if(count==2){ // 2 is used bcz there are only 2 factors of prime number
            return "YES";
          }
          else{
            return "NO";
          }
        };
Input 1:  7
Output 1:  YES
Input 2:  4
Output 2: NO
